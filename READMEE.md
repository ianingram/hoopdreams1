🏀 Hoop Dreams

The Story of the Player. Hoop Dreams is a decentralized ecosystem and responsive web application engineered to bridge the structural gap between impromptu local park basketball games and the professional circuit. It captures the intrinsic value of streetball by transforming physical interactions into a verifiable digital ledger of performance, data, and monetization.

🌟 Key Features

Dynamic Player Avatars & NFT Identity: Fully customizable SVG-based player avatars (skin tone, hair style, jersey color) that act as the visual representation of a player's on-chain NFT card.

Decentralized Tokenomics: Integrated Web3 wallet hub tracking Solana (SOL), Ethereum (ETH), and proprietary Hoop Dreams (HD) tokens for match staking and payouts.

Advanced Attributes Matrix: Detailed tracking of player metrics including Aura, Riz, Mental Toughness, Dunking, and Court Awareness.

Match Orchestration & Zone Tracking: Interactive location selection (by Zip Code or State/County) to find live local courts, view interactive maps, and join active stakes.

Sideline Observer Mode: Dynamic substitution mechanics allowing sideline players to sub into live matches, record clips, and log statistics.

Weekend Warriors Program: Active tournament recruitment system for high-stakes weekend events.

🚀 Tech Stack

Frontend Framework: React.js

Styling: Tailwind CSS (Fully responsive mobile-first design)

Icons & UI: Lucide React

Build Tool: Vite

🛠️ Getting Started

To run this project locally:

Clone the repository:

git clone [https://github.com/your-username/hoop-dreams.git](https://github.com/your-username/hoop-dreams.git)
cd hoop-dreams


Install dependencies:
Make sure you have Node.js installed, then run:

npm install


Start the development server:

npm run dev


Open your browser and navigate to the local host address provided in the terminal (usually http://localhost:5173).
